# AuthFlow - Modern Authentication System

A production-ready, secure authentication system built with Django and TailwindCSS.

## Features

- ✅ Email-based authentication (no username)
- ✅ OTP verification with auto-submit
- ✅ Rate limiting and security features
- ✅ Modern dashboard with sidebar
- ✅ Password reset flow
- ✅ Responsive design
